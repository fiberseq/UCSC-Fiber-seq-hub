Description

Fiber-seq Inferred Regulatory Elements (FIRE)

This track overlays Fiber-seq percent-accessible chromatin for a panel of commonly used cell lines. If you are unfamiliar with Fiber-seq please see the references below for a detailed description, but in short it is useful to think of it as a long-read, single-molecule, haplotype-resolved version of DNaseI/ATAC-seq that can be used to identify regions of chromatin accessibility. For the full per-sample, per-track-type compendium (peaks, individual haplotypes, and every additional sample), see the Fiber-seq Compendium track.

Questions or problems with this hub? Contact Mitchell Vollger (mrvollger@genetics.utah.edu) or Andrew Stergachis (absterga@uw.edu).

Display Conventions

This track is on by default and overlays percent-accessible chromatin for GM12878, K562, HepG2, H1, Hap1, Hek293T, and Jurkat, each drawn in a different color, as semi-transparent overlapping lines so regions of shared vs. cell-line-specific accessibility are visible at a glance.

Track Descriptions

  • Percent accessible: The percent of fibers that contain a FIRE element (a Fiber-seq-inferred regulatory element) overlapping a given position in the genome, combining both haplotypes.

    • Methods

    FIREs are MTase sensitive patches (MSPs) on Fiber-seq reads that are inferred to be regulatory elements on single chromatin fibers, identified using semi-supervised machine learning (the Mokapot framework and XGBoost). Every individual FIRE element is associated with a precision value, which indicates the probability that the FIRE element is a true regulatory element. Please refer to https://fiberseq.github.io/ for more details.
