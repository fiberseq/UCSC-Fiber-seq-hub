Description

Fiber-seq Inferred Regulatory Elements (FIRE)

This track overlays single-molecule chromatin fiber sequencing (Fiber-seq) FIRE percent-accessible chromatin signal for a panel of commonly used cell lines. Fiber-seq is a long-read, single-molecule, haplotype-resolved chromatin stenciling method for epigenomic mapping. For the full per-sample, per-track-type compendium (peaks, individual haplotypes, and every additional sample), see the Fiber-seq Compendium track.

Questions or problems with this hub? Contact the Vollger Lab (mrvollger@genetics.utah.edu) or the Stergachis Lab (absterga@uw.edu).

Display Conventions

This track is on by default and overlays percent-accessible chromatin for GM12878, K562, HepG2, H1, Hap1, Hek293T, and Jurkat cells, each drawn in a different color, as semi-transparent overlapping densities so regions of shared vs. cell-line-specific accessibility are visible at a glance.

Track Descriptions

  • Percent accessible: The percent of fibers that contain a FIRE element (a Fiber-seq-inferred regulatory element) overlapping a given position in the genome, combining both haplotypes.

    • Methods

    FIREs are Modification Sensitive Patches (MSPs) on Fiber-seq reads that are inferred to be regulatory elements on single chromatin fibers, identified using semi-supervised machine learning (Vollger, Swanson et al.). Every individual FIRE element is associated with a precision value, which indicates the probability that the FIRE element is a regulatory element. Please refer to https://fiberseq.github.io/ for more details.
